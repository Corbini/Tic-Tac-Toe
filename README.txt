Tic-Tac-Toe

Purpose:
    You will learn how to work with Python's time and math modules as well as get continual practice with nested if statements. Make gui with PyQT 6 (newest).

Quest:
   1. Make a map that print in terminal
   2. Make a player input
   3. Make a computer input
   4. Make game logic
   5. Show window on screen
   5. Make map in window
   6. Make user input on window
   7. Make start game button. endgame status shown on screen
   
  
Dane:
   time https://docs.python.org/3/library/time.html
   math https://docs.python.org/3/library/math.html
   PyQT https://riverbankcomputing.com/software/pyqt/intro
